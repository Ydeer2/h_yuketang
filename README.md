## 脚本简单介绍

脚本使用JavaScript编写，不需要依赖任何环境，直接在网页中运行即可。

此外，脚本开发用的测试网页为雨课堂（蓝色版），其他服务器的雨课堂，因为条件有限没有测试，不清楚是否适用（如果页面都是一样的话就可以使用）。

![image-20260621170420135](https://cdn.jsdelivr.net/gh/Ydeer2/Images@main/image-20260621170420135.png)

## 脚本的使用

进入雨课堂后，直接点击对于课堂，进入对应的视频页面，如下
![image-20260621171231359](https://cdn.jsdelivr.net/gh/Ydeer2/Images@main/image-20260621171231359.png)打开网页控制台，复制js代码，粘贴到控制台中



![image-20260621171613517](https://cdn.jsdelivr.net/gh/Ydeer2/Images@main/image-20260621171613517.png)

点击回车即可运行

![屏幕截图 2026-06-21 171646](https://cdn.jsdelivr.net/gh/Ydeer2/Images@main/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-06-21%20171646.png)

## 注意事项：

游览器选择：尽量使用谷歌游览器，支持控制台粘贴代码的游览器

虽然在脚本中实现了二倍数播放功能，但是实际情况中无法稳定使用，大部分时候是1倍数播放

脚本会自动跳过已经完成的视频，第二次使用脚本，可以在第一个视频处（已完成）开始粘贴代码。
